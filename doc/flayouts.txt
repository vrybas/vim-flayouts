INTRODUCTION                                    *flayouts*

Flayouts.vim aims to ease work with [Fugitive](github.com/tpope/vim-fugitive)
by providing window layouts for certain operations, such as:

  * Commit
  * Pull Request review
  * Conflict Resolution

COMMANDS                                        *flayouts-commands*


                                                *flayouts-:Glstatus*
:Glstatus             Open new tab with three-window layout. Top window
                      contains `:Gstatus`, bottom window contains current
                      file, right window contains output of git-diff.

                                                *flayouts-:Glcommit*
:Glcommit             Perform a commit from *:Glstatus* window, and close tab.

                                                *flayouts-:Glabort*
:Glabort              Cancel a commit from *:Glstatus* window, and close tab.

                                                *flayouts-:GlpullRequest*
:GlpullRequest        Open new tab with two-window layout. Left window contains
                      current file, right window contains git-request-pull
                      output (summary of all commits between 'master' and
                      current branch).

                                                *flayouts-:GlresolveConflict*
:GlresolveConflict    Open new tab with three-window layout. Top window
                      conains 'ours' version of current file. Middle window
                      contains current file with conflict maks. Bottom window
                      contains 'theirs' version of current file.

                                                *flayouts-:GlprDiff*
:GlprDiff             Generate output of 'git-request-pull' into a temp file,
                      and shows this temp file in current window.

                                                *flayouts-:GlopenFromDiff*
:GlopenFromDiff       Open file containing code chunk under cursor from any
                      generated 'diff' output, and run `:Gblame`. Especially
                      useful from `:GlpullRequest` tab.

OPTIONS                                         *flayouts-options*

<
let g:flayouts_base_branch = "origin/master"
<
Default: "origin/master"

Used for :GlpullRequest and :GlprDiff to run "requiest-pull" against specified
branch.
